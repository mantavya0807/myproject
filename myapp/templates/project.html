SecureWater
Previous Next
Overview

SecureWater is an advanced image watermarking system that helps protect digital assets using frequency-domain techniques. The system provides robust watermarking with tamper detection capabilities, ensuring the integrity and authenticity of digital images.

Features
  • Invisible watermark embedding using DCT/DWT techniques
  • Tamper detection and verification system
  • User authentication and image ownership tracking
  • Robust against image modifications and compression
  • Public verification portal for watermark checking
Technologies Used

React, Node.js, MongoDB, Sharp image processing library, JWT for authentication, Express.js for API development
